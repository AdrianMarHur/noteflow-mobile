# NoteFlow Mobile

NoteFlow es una aplicación móvil de productividad que reúne en un único lugar los tres pilares de la organización personal: **capturar, procesar y ejecutar**.

- 📝 **Notas** — captura información en bruto al instante
- 💡 **Ideas** — organiza y estructura lo que has capturado
- ✅ **Tareas** — convierte tus ideas en acciones concretas

## 🚀 Stack tecnológico

| Tecnología | Uso |
|------------|-----|
| React Native + Expo | Base de la aplicación |
| TypeScript | Tipado estático |
| Expo Router | Navegación |
| Gluestack UI | Sistema de diseño |
| Zustand | Estado global |
| FlashList | Listas de alto rendimiento |
| AsyncStorage | Persistencia local |
| Zod | Validación de formularios |
